# Roles and Skills Analysis

## Roles for components

Based on the architecture of **EduPlatform**, the following roles are required:

* **Web Frontend (SPA)**
    * **Frontend Engineer:** Responsible for the UI/UX and interacting with the API.
* **Backend API & Auth Service**
    * **Backend Engineer (Python):** Designs the REST API, implements business logic, and optimizes database queries.
* **Database (PostgreSQL)**
    * **Database Administrator (DBA):** Manages schemas and migrations. (Often handled by the Backend Engineer in small teams).
* **Infrastructure**
    * **DevOps Engineer:** Manages Docker containers, Nginx, and cloud deployment.

## Common skills across roles
* **Git & GitHub:** For version control and collaboration.
* **HTTP/REST:** Understanding API communication.
* **Linux Terminal:** Basic server management.
* **English:** Reading documentation.
